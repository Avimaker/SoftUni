Text Input

The input element is used to create interactive controls for web-based forms in order to accept data from the user; a wide variety of types of input data and control widgets are available, depending on the device and user agent.

Входният елемент input се използва за създаване на интерактивни контроли за уеб базирани формуляри, за да се приемат данни от потребителя; голямо разнообразие от типове входни данни и контролни модули са налични в зависимост от устройството и потребителския агент.

The input element is one of the most powerful and complex in all of HTML due to the sheer number of combinations of input types and attributes.

Елементът input е един от най-мощните и сложни в целия HTML поради големия брой комбинации от типове вход и атрибути.

Attribute - type="text"

The default value. A single-line text field. Line-breaks are automatically removed from the input value.

Стойността по подразбиране. Едноредово текстово поле. Прекъсването на редовете се премахва автоматично от въведената стойност.

<input type="text" id="username" name="username" placeholder="Username">

Attribute - type="password"

A single-line text field whose value is obscured. Will alert user if site is not secure.

Едноредово текстово поле, чиято стойност е скрита. Ще предупреди потребителя, ако сайтът не е защитен.

<input type="password" id="password" name="password" placeholder="Password">

Attribute - type="number"

A control for entering a number. Displays a spinner and adds default validation. Displays a numeric keypad in some devices with dynamic keypads.

Контрола за въвеждане на число. Показва спинер и добавя проверка по подразбиране. Показва цифрова клавиатура в някои устройства с динамични клавиатури.

<input type="number" id="number" name="number" value="5">

Attribute - type="email"

A field for editing an email address. Looks like a text input, but has validation parameters and relevant keyboard in supporting browsers and devices with dynamic keyboards.

Поле за редактиране на имейл адрес. Изглежда като въвеждане на текст, но има параметри за валидиране и подходяща клавиатура за поддържане на браузъри и устройства с динамични клавиатури.

<input type="email" id="email" name="email" placeholder="email">

Attribute - type="search"

A single-line text field for entering search strings. Line-breaks are automatically removed from the input value. May include a delete icon in supporting browsers that can be used to clear the field. Displays a search icon instead of enter key on some devices with dynamic keypads.

Едноредово текстово поле за въвеждане на низове за търсене. Прекъсването на редовете се премахва автоматично от въведената стойност. Може да включва икона за изтриване в поддържащи браузъри, която може да се използва за изчистване на полето. Показва икона за търсене вместо клавиш за въвеждане на някои устройства с динамични клавиатури.

<input type="search" id="search" name="search" placeholder="Search ...">




Textarea

The textarea element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

Елементът textarea представлява многоредова контрола за редактиране на обикновен текст, полезна, когато искате да позволите на потребителите да въвеждат значително количество текст в свободна форма, например коментар към рецензия или формуляр за обратна връзка.

<textarea name="textarea" rows="5" cols="45">Text goes here</textarea>




Radio

A radio button, allowing a single value to be selected out of multiple choices with the same name value.

Бутон за избор radio, позволяващ да бъде избрана една стойност от множество възможности за избор със същата стойност на име.

Attribute - type="radio"

<p>
    <input type="radio" name="fruits" id="apple">
    <label for="apple">Apple</label>
</p>

<p>
    <input type="radio" name="fruits" id="pear">
    <label for="pear">Pear</label>
</p>

<p>
    <input type="radio" name="fruits" id="banana">
    <label for="banana">Banana</label>
</p>

Check box , позволява избиране/демаркиране на единични стойности.

Attribute - type="checkbox"

Select

The select element represents a control that provides a menu of options.

Елементът select представлява контрола, която предоставя меню с опции.

<label for="pet-select">Choose a pet:</label>
<select name="pets" id="pet-select">
    <option value="">--Please choose an option--</option>
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
    <option value="hamster">Hamster</option>
    <option value="parrot">Parrot</option>
    <option value="spider">Spider</option>
    <option value="goldfish">Goldfish</option>
</select>




Multiselect

This Boolean attribute indicates that multiple options can be selected in the list. If it is not specified, then only one option can be selected at a time. When multiple is specified, most browsers will show a scrolling list box instead of a single line dropdown.

Този булев атрибут показва, че в списъка могат да бъдат избрани множество опции. Ако не е посочено, тогава може да бъде избрана само една опция в даден момент. Когато е посочено множество, повечето браузъри ще показват превъртащо се списъчно поле вместо падащо меню с един ред.

<label for="pet-select">Choose a pet:</label>
<select name="pets" id="pet-select" multiple>
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
    <option value="hamster">Hamster</option>
    <option value="parrot">Parrot</option>
    <option value="spider">Spider</option>
    <option value="goldfish">Goldfish</option>
</select>

Fieldset

The fieldset element is used to group several controls as well as labels within a web form.

Елементът fieldset се използва за групиране на няколко контроли, както и етикети в уеб формуляр.

<fieldset>
    <legend>Login</legend>

    <p class="field">
        <label for="username">Username</label>
        <input type="text" name="username" id="username">
    </p>

    <p class="field">
        <label for="password">Username</label>
        <input type="password" name="password" id="password">
    </p>

    <input type="submit" value="Login">

</fieldset>
